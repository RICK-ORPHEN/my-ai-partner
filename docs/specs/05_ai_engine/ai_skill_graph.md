# AI Skill Graph

AI School Platform

This document defines the AI Skill Graph used to structure and categorize AI-related skills within the platform.

The AI Skill Graph organizes skills into a hierarchical structure that allows:

- skill evaluation
- talent discovery
- learning path generation
- category ranking

The structure follows a tree model.

---

# 1 Skill Graph Structure

The skill graph is organized in four levels.

Level 1: Domain
Level 2: Skill Category
Level 3: Skill
Level 4: Sub Skill

Example Structure

AI Development
    ├ Agent Systems
    │     ├ Multi-Agent Systems
    │     ├ Task Automation Agents
    │     └ API Agents
    │
    ├ AI Application Development
    │     ├ AI SaaS Development
    │     ├ Workflow Automation
    │     └ API Integration
    │
    └ AI Data Processing
          ├ Data Pipelines
          ├ AI Data Analysis
          └ Data Automation

# 9 Skill Level System

Each skill node has a level from 0 to 5.

Level Meaning

0 Not Learned
1 Basic Awareness
2 Basic Usage
3 Practical Usage
4 Advanced Implementation
5 Expert

---

# 10 Skill Score Calculation

Each skill contributes to the AI Skill Score.

Skill Score Formula

Skill Score =
Skill Level × Skill Weight

Example

Skill: AI Agent Systems
Level: 3
Weight: 40

Score = 120
