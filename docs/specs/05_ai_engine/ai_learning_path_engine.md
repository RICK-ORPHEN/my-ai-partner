# AI Learning Path Engine

AI School Platform

This document defines the AI Learning Path Engine that generates personalized learning paths for users.

The system analyzes a user's current skills, goals, and activity history to recommend the next optimal learning steps.

The engine is designed to accelerate skill development and improve talent matching for companies.

---

# 1 Purpose

The AI Learning Path Engine provides:

- personalized learning paths
- automatic curriculum recommendations
- skill gap analysis
- career development guidance

This system helps users efficiently progress from beginner to professional level.

---

# 2 Core Inputs

The engine analyzes the following data.

User Skill Score
User Skill Graph
Reputation Score
Completed Courses
Project Experience
Learning Goals

Example:

User Goal
"AI Agent Developer"

Current Skills

Prompt Engineering Level 3
API Integration Level 2
Automation Systems Level 1

---

# 3 Skill Gap Analysis

The engine compares the user's current skills against the target skill profile.

Example Target Skill Profile

AI Agent Developer

Prompt Engineering Level 4
API Integration Level 4
Automation Systems Level 3
Agent Systems Level 3

Skill Gap Output

Prompt Engineering → +1 level needed
API Integration → +2 levels needed
Automation Systems → +2 levels needed
Agent Systems → +3 levels needed

# 7 Dynamic Path Adjustment

Learning paths update dynamically based on progress.

When a user:

completes a course
improves a skill score
finishes a project

the engine recalculates the optimal next step.

---

# 8 Skill Acceleration Detection

The system detects rapid skill improvement.

Example

If a user demonstrates high skill level in projects, the engine may skip beginner courses.
