## 横断スキルコース（任意 + 全業種で利用可）

1. Claude Skills とは何か / なぜSkillが強いか
2. SKILL.md構造（description triggers, body, references）
3. 自社業務を解析しSkillの守備範囲を決める
4. プロンプト＋テンプレ設計
5. references/ サブファイル / バリエーション
6. テスト・評価（skill-eval）
7. プラグイン化＋marketplace公開

### B. LP制作コース（lp_design）
1. LP=コンバージョン装置 / 構造の基本
2. ヒーロー設計／一次情報の発掘
3. デザインシステム（lp-XX 適用）
4. コピーライティング × AI
5. 画像生成（imagegen） × ヒーロー画像
6. レスポンシブ調整
7. 公開（Vercel or Squarespace） × 計測

### C. 企画書・提案書コース（proposal_design）
1. 企画書の役割と構造（Why/What/How/Risk）
2. オーディエンス分析
3. 構成テンプレ：俯瞰 → 課題 → 解決策 → 計画
4. AIで一次ドラフト（auto-proposal/upe-v1）
5. 図解（mermaid/フロー）
6. 数値根拠（市場サイズ・ROI）
7. 配布形式（PPTX/PDF/HTMLページ）

### D. 自動オペレーションチャットツールコース（automation_chat）
1. なぜチャットツールが業務を激変させるか
2. プラットフォーム比較（Slack/Telegram/LINE/Discord）
3. ボット骨格（Webhook → Bot → 返信）
4. AI連携（Keyless AI でAPI連携）
5. 業務テンプレ（FAQ / 売上速報 / 予約管理）
6. データベース連携（Supabase）
7. 運用＆監視
