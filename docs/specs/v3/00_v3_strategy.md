# AI School v3 — 汎用プロダクト開発スクール

## 方針転換（v2 → v3）

| 項目 | v2 | v3 |
|------|----|----|
| 対象 | 飲食特化 | 8業種 + 横断スキル3 |
| 売り | 業務課題を解ける思考力 | **作って公開できる実行力** |
| 終点 | レッスン完走 | **公開済プロダクト + ポートフォリオ** |
| 公開 | 想定なし | Vercel/Supabase or Squarespace（学習者レベル分岐） |
| マネタイズ | 受講料のみ | 受講料 + アフィリエイト（Vercel/Supabase/Squarespace） |

## 北極星指標
受講生がプロダクトを **公開URLとして所有**し、ポートフォリオに掲載されている状態をMVP完成と定義する。

## 8業種 × 7レッスン
飲食 / 小売 / 不動産 / 医療 / 士業 / 建設・製造 / 美容 / 教育

各業種で以下の7段：
1. AI活用入門（業種特有のユースケース）
2. プロンプト基礎
3. データ整理・自動化（スプレッドシート × AI）
4. 顧客対応自動化（チャットボット）
5. コンテンツ生成（LP/SNS/メール）
6. **業務システム制作**（自分のサービスを作る）
7. **公開・運用**（Vercel/Supabase or Squarespace 契約 → 公開）

## 横断3スキルコース（全業種共通・任意受講）
- **A. スキル開発コース**（Claude Skills の作り方 → 自社用Skillを作る）
- **B. LP制作コース**（コピー設計 → デザイン → 公開）
- **C. 企画書・提案書コース**（ロジカル構成 → 図解 → PPTX出力）
- **D. 自動オペレーションチャットツールコース**（Telegram/Slack bot → API連携）

## 公開ルート分岐（Onboarding時に判定）

### Track A — コーディング志向（Vercel + Supabase）
- 推奨対象: 「Web開発を学びたい / 技術寄り」を選択した受講生
- カリキュラム後半でNext.jsベースのテンプレ提供
- アフィリ: Vercel Pro / Supabase Pro 契約誘導

### Track B — ノーコード志向（Squarespace）
- 推奨対象: 「コードは書きたくない / デザイン寄り」を選択した受講生
- Squarespace上のテンプレ + AI生成コンテンツで完成
- アフィリ: Squarespace Business 契約誘導

## ポートフォリオ機能
- 受講生が完成させた成果物を `portfolio.aischool.app/{username}` で自動公開
- AI レビュー機能：完成度スコア / 改善点 / 次のレベルへの示唆
- 卒業証明としてPDF認定証も発行
