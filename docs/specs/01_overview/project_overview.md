# AI School Platform

Project Overview

This document describes the overall concept of the AI School Platform.

The platform is an AI education and talent ecosystem that connects learning, evaluation, and real-world opportunities.

---

# Core Concept

Education → Skill Evaluation → Talent Database → Corporate Opportunities

The system is designed to develop AI-native professionals who can create value using artificial intelligence.

---

# Platform Components

The platform consists of three major systems:

1. Public Website (Marketing & Onboarding)
2. User Learning Dashboard (LMS Platform)
3. Admin Management Panel

---

# Target Users

## Individuals

People who want to develop practical AI skills for:

- career advancement
- freelance work
- entrepreneurship
- AI product development

---

## Companies

Organizations that want to:

- train internal AI talent
- improve productivity
- build AI-driven operations
- reduce outsourcing

---

# Platform Vision

To build the largest AI talent ecosystem connecting education, evaluation, and real-world opportunities.

---

# Key Platform Modules

Public Website
Learning Platform (Dashboard)
Skill Evaluation System
Talent Database
Corporate Opportunity System

---

# Technology Goals

The platform should support:

- scalable user learning
- talent evaluation
- corporate talent discovery
- subscription-based monetization
