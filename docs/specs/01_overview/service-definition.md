# サービス定義 / Service Definition

## 1. 正体
本サービスは「AI講座サイト」ではない。
**業種 × 目的 × 現在地** を起点に、受講者ごとに必要な課題・思考・実装テーマを出し分け、ヒント中心で考えさせながら、最終的に **実務システムを自力で構築できる状態** まで導く、インタラクティブ型 AI 実践スクール OS である。

## 2. 対象ユーザー
- 業種特化で AI を導入したい中小〜中堅企業の担当者
- 社員に AI 開発を内製化させたい経営者
- Cowork / ChatGPT / Gemini を使って自社業務を自動化したい個人
- ノーコード / ローコードで実務システムを作りたい非エンジニア

## 3. 提供価値
1. 業種・目的別に「何から学ぶべきか」が自動で提示される
2. 問いとヒントで思考を誘導し、答えを配らない
3. GPT-5.4 が提出物を評価し、足りない視点を指摘する
4. 最終アウトプットが「知識」ではなく「稼働する実務システム」になる

## 4. やること
- カリキュラム生成（業種×目的×レベル×制約）
- 問いとヒントによる思考誘導
- 実装環境（Cowork / Claude / Gemini）への橋渡し
- 提出物の AI 評価と改善フィードバック
- 業種別 AI 人材データベース（マッチング）

## 5. やらないこと
- 完成コード・完成テンプレートの配布
- 受講者の代わりに設計を全部行う
- 一般的な AI リテラシー座学の販売
- 別世界観の独立サイト構築（既存 my-ai-partner の下で拡張）

## 6. 成功条件
- 初見ユーザーが「何を学べるか / 自分に合うか / どう始めるか」を迷わない
- オンボーディングから最初のレッスンまで一気通貫で入れる
- チャットbotコースが1本、完成形として成立している
- AI の役割が「答え配布」ではなく「思考支援」になっている
- 他コースへ横展開できる骨格ができている
