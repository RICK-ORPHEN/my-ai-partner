# ビジネスモデル / Business Model

## 1. 収益の柱
1. **スクール事業**（メイン・初期）
   - 法人向けサブスク: ¥98,000 / 社 / 月（10名まで）
   - 個人向けサブスク: ¥9,800 / 月
   - 企業研修パック: ¥500,000〜 / 回
2. **マッチング手数料**（スケール後）
   - 受講修了者を業種別 AI 人材として企業と接続
   - 初回成約: 想定年収の 15%
   - 月額顧問: 月額の 10%
3. **コンテンツライセンス**
   - 業種団体・商工会議所への教材ライセンス供与

## 2. コスト構造
- AI API（GPT-5.4 / Gemini）
- Supabase / Vercel / GitHub
- マーケティング（SNS / SEO / セミナー）
- コンテンツ制作
- CS / 運営

## 3. ユニットエコノミクス（想定）
- 法人1社 MRR: ¥98,000
- 変動コスト（AI + インフラ）: ¥15,000
- 粗利率: 約 85%
- CAC 目標: ¥200,000
- LTV 目標: ¥1,500,000（15ヶ月）

## 4. 2年でのバイアウトシナリオ
- ベンチマーク: Aidemy ¥5.4億 売上 → Accenture ¥60億 買収（PSR 11.1x）
- 24ヶ月目標 ARR: ¥7.7億
- Exit 想定額: ¥62〜92億（PSR 8〜12x）

## 5. 差別化ポイント
- 業種×目的×レベルで出し分ける動的カリキュラム
- 答えを配らないヒント型 AI 評価
- 修了者が実務システムを稼働させている状態
- マッチング事業による修了後の収益接続
