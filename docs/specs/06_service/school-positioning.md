# ポジショニング v2 — 飲食特化版

> 2026-04-11 改定。「AIツールを学ぶ」→「業務課題を自分で解ける思考力」に転換。

## 1. ひと言定義
> **「社員がAIで業務課題を解ける飲食店になる」**

ツール操作ではなく思考力を売る。Cowork は実践環境であり、商品ではない。

## 3. 差別化の核
1. **思考力を売る**: ツールが変わっても残る力
2. **飲食特化**: 予約・問い合わせ・シフトなど具体的な課題
3. **答えを渡さない**: ヒント×AI評価で成長させる
4. **成果物が残る**: 修了後に現場で稼働するシステム

## 4. 初期ターゲット
- 飲食店オーナー（個人店〜10店舗規模）
- 飲食チェーンの DX 担当者
- 飲食業向け IT 支援を行うコンサルタント

## 5. メッセージング（飲食特化版）
- オーナー向け: 「予約電話が半分になる。社員が自分で作るから外注費もゼロ」
- 店長向け: 「同じ質問に何度も答えなくていい。botに任せて接客に集中」
- 本部向け: 「各店舗のAI活用度をスコアで可視化」
